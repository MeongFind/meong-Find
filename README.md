# meong-Find

## 프로젝트 소개

반려동물 1500만 시대, 반려견에 대한 실종 제보와 도움을 청할 곳은 없는 상황

이러한 실종된 반려동물을 보고 신고하거나 도움이 필요한 주인들에게 소식을 알려 강아지가 주인의 품으로 돌아갈 수 있도록 만든 서비스입니다.

## 지원기능

1. 로그인
2. 회원가입
3. 글 등록
4. 글 수정/삭제
5. 댓글 등록
6. 댓글 수정/삭제
7. 검색
8. 채팅
